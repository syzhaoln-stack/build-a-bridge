# 统一术语表

| 概念 | 统一写法 | 说明 |
|---|---|---|
| generative AI | 生成式 AI | 首次出现可写“生成式人工智能（生成式 AI）” |
| large language model | 大语言模型（LLM） | 不用“大模型”代替所有 AI 系统 |
| artificial intelligence skill | AI Skill | 指可复用的代理工作流时使用；普通能力写“AI 应用能力” |
| game-based learning | 游戏化学习/游戏式学习 | 若仅有积分、排名，写“游戏化机制”；若以完整游戏为学习环境，写“游戏式学习” |
| feedback loop | 学习回路 | 固定为“尝试—反馈—修改—再解释” |
| evidence chain | 证据链 | 包括来源、假设、过程、核验、版本与反思 |
| judgment | 专业判断 | 不泛写“判断力”，需说明判断对象和依据 |
| visualization | 可视化 | 区分示意动画、数据图和经计算验证的工程模型 |

全文避免把“AI”“ChatGPT”“大语言模型”“代理”混作同一概念。具体产品名只在案例复现需要时出现，方法章节尽量用能力描述，降低时效性损耗。
