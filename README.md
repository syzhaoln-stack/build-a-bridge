# 请把手弄脏：AI时代的学与玩，或一座桥的哲学

> **120万字课堂录音，2年教学实验，1个核心问题：当机器能比人更快地学会一切，人为什么还要学习？**

这不是一本"AI工具使用指南"。这是一位工科教师用桥梁思维重新设计课堂的哲学记录。

## 在线体验

[SpanCraft 桥造计划](https://syzhaoln-stack.github.io/spancraft-bridge-game/) —— 书中提到的桥梁建造游戏，你可以在线搭一座桥。

## 目录

|------|------|---------|
| 序言 | 在没人爱土木的时代，我给一座桥写了封情书 | 为什么要写这本书？ |
| 一：两岸 | AI如何学习，人如何学习 | 两种"学习"的本质区别 |
| 二：荷载 | 新范式撞击旧系统 | 什么在冲击我们的教育？ |
| 三：传力路径 | 学习回路的设计 | 学习如何真正发生？ |
